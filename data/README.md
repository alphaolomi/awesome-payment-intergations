# Data Catalogue

This directory contains machine-readable records for the project catalogue.

## Layout

- `mobile-money/`: mobile money providers and direct mobile wallet rails
- `psp/`: payment service providers, aggregators, and bank gateways
- `sdk/`: SDK and integration libraries

## Authoring rules

1. Use YAML (`.yml`) files.
2. Follow the normalized schema in `SCHEMA.md`.
3. Use lowercase kebab-case for file names.
4. Set `maintainer_type` as `official`, `community`, or `unknown`.
